# vst-out
An Ableton Extension for discovering, organizing, and exporting all locally installed VST plugins (VST 3, AU, and other formats) from a single interface.

A minimal Ableton Live Extension for scanning and exporting your entire set of local installed VSTs (VST 3, AU, VST 2, and more) in one organized list. Discover what you have, organize by category or tag, and export to multiple different file types (JSON, CSV, TXT, MD) for reference or archival.

### Features:

- Automatic scanning of system VST install directories
- Export plugin inventory as JSON, CSV, TXT, or MD
- Cross-platform support (macOS/Windows)

### Use Cases:

- Quickly audit what plugins you have
- Generate a list for documentation or backup
- Identify unused VSTs
